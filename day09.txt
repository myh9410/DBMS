# VMWare 복제 종류
 - linked clone
	장점 : 하드 제외하고 원래 있던 걸 복사해서 용량 차지가 적음
	단점 : 성능이 저하됨. 
	       원본 가상 컴퓨터와 복제된 가상 컴퓨터가 항상 같이 있어야 함.
 - full clone
	장점 : 성능이 저하되지 않음
	       원본 가상컴퓨터와 복제된 가상컴퓨터가 따로 있어도 문제 되지 않는다.
	단점 : 용량을 많이 차지함.(용량이 낭비됨.)

#SSH(Secure Shell) : 원격지에 있는 장비로 접속하는 서비스.
			접속한 장비에 명령어를 실행시킬 수 있음.
사용 목적 : 원격지에 있는 장비를 관리하기 위함.
ex) 우리 집에서 회사에 있는 장비로 접속. 회사 장비의 명령어를 수행시킬 수 있음.
SSH이용해서 원격지 장비에 접속. 서버실의 DB서버로 사무실에서 SSH로 접속해서 장비 관리.
