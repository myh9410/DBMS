#데이터베이스 : 데이터가 저장되어지는 공간
#DBMS -Database Management System

#DBMS : 데이터베이스를 관리 및 제어하는 프로그램
             데이터를 저장, 수정, 조회, 삭제 등을 해주는 프로그램

#DBMS 프로그램 종류 : MySQL, MariaDB, ORACLE, MongoDB, DB2 등등
수업 -> MySQL로 진행. 한국에서는 ORACLE 제일 많이 씀.

#SQL : DBMS에 어떤 작업을 하라고 명령할 때 사용하는 문구
	ex) 데이터 저장, 데이터 삭제 등을 지시할 때 사용

#표준 SQL : 국제표준화 기구에서 전 세계적으로 데이터베이스 프로그램에서 사용하는 SQL의 표준을 제안하고 있음..

#표준을 정해두는 이유 : 과거에 서로 다른 회사 끼리는 데이터를 주고받지못함 = 호환이안됨. 서로 다른 회사끼리 통신이 되도록 하기위해. ex) 삼성 갤럭시와 아이폰이 서로 문자를 주고 받을 수있음.
