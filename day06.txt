# 입력모드 : 무엇인가를 작성할 수 있는 모드임

# 입력모드 상태에서 esc를 누르면 입력 모드를 빠져나감

# vi를 통해 파일을 열었을 때 이동모드 상태에서 시작 됨
# 이동모드는 파일을 이리 저리 커서를 왔다갔다할 수 있는
  모드임


# 입력 모드 진입 단축키
	i : 현재 커서가 위치 해 있는 곳에 왼쪽으로 진입
	a : 커서의 오른쪽으로 입력 모드 진입
	I : 커서가 있는 행에 처음으로 입력 모드
	    진행...
	A : 커서가 있는 행에 마지막으로 입력모드 진행
	o : 커서가 있는 행 아래로 입력모드 진입
	O : 커서가 있는 행 위로 입력모드 진입
	R : 커서의 좌측으로 입력 모드를 진입시키는데
	    내용을 작성하면 원래 있는 글자를 덮어쓰기
	    하면서 작성됨

# 이동을 하는 단축키
	^ : 커서가 있는 행에 가장 처음으로 이동
	$ : 커서가 있는 행에 가장 마지막으로 이동
	[[ : 커서를 문서에 처음으로 이동
	]] : 커서를 문서에 마지막으로 이동

	h : 커서를 한 칸 왼쪽으로 이동
	l : 커서를 한 칸 오른쪽으로 이동
	j : 커서를 한 칸 아래로 이동
	k : 커서를 한 칸 위로 이동

# 삭제 관련 단축키
	x : 커서가 위치 해 있는 글자를 삭제
	12x : 커서가 있는 글자를 포함해서 12글자 삭제
	dd : 커서가 있는 행을 삭제
	8dd : 커서가 잇는 행을 포함해서 8개의 행을 삭제


	u : 이전에 한 작업을 취소(되돌리기)
	Ctrl + r : 되돌리기한 것을 취소

	v : 블럭모드를 진입

# 블럭모드 : 블럭을 씌울 수 있는 모드



# 복사 관련 단축키
	yy : 커서가 있는 행을 복사
	p : 커서의 아래 행으로 붙여넣기
	P : 커서의 위에 행으로 붙여넣기
	5yy : 커서가 있는 행을 포함해서 5개의 행을 복사
	:3,5y : 3번 행부터 5번 행까지 복사하기
	:.,6y : 커서가 위치한 행부터 6번 행까지 복사하기
	:2,.y : 2번 행부터 커서가 위치한 행까지 복사하기

# 검색 및 치환
	/lg : lg라는 문자열을 검색
	(검색을 진행 하고 n을 누르면 검색 된 것 중에
	다음꺼로 이동하고, N을 누르면 이전꺼로 이동)

	:s/lg/LG <-- 커서가 위치한 행에 lg를 LG로 바꾸는
		     데, 한개만 바꾸기...
	:s/lg/LG/g <-- 커서가 위치한 행에 lg를 LG로 바꾸
			데, 전부 다 바꾸기...
	:3,5s/lg/LG <-- 3번 행부터 5번 행에 lg를 LG로 
			바꾸기...
	:%s/lg/LG <-- 문서전체에 lg를 LG로 바꾸기
	:%s/samsung/&/g <-- 문서전체에 samsung이라는
			문자열이 몇 개 있는 지 검색하는
			용도로 씀...

	:set nonu : 행 번호 붙인거 취소하기
